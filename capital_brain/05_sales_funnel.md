## 2. 各層の詳細設計

### Layer 1: TOFU（Top of Funnel）— X運用

#### 目的
- ブランド認知の拡大
- ターゲット層へのリーチ
- noteへの流入経路の確立

#### KPI

| 指標 | 目標（月間） |
|------|------------|
| インプレッション | 100万 |
| フォロワー増加 | +1,000/月 |
| プロフィールクリック | 5,000 |
| リンククリック（note/LINE） | 1,000 |

#### 導線設計

```
Xポスト
  ├→ プロフィール訪問
  │    ├→ プロフリンク → note記事（無料）
  │    └→ プロフリンク → LINE登録
  │
  ├→ ポスト内CTA → note記事（直接リンク）
  │
  └→ リプライ/引用RT → エンゲージメント → フォロー
```

#### Xプロフィール設計

```
名前: Capital Brain｜AI×資本家脳
ID: @capital_brain

Bio:
AIで「資本家の脳」をインストールする💡
労働者脳→資本家脳へのOSアップデート
AI×資産形成×収益化の3本柱で発信
▼ 無料note & LINE限定特典はこちら ▼

リンク: lit.link/capitalbrain
（lit.linkにnoteとLINEの導線を設置）

固定ポスト: 最も反応の良い無料note記事への導線
```

## 6. リスク管理

| リスク | 対策 |
|--------|------|
| noteプラットフォーム依存 | LINE/メルマガで自社リスト構築 |
| X アカウント凍結リスク | 規約遵守・複数チャネル運用 |
| 投資助言規制 | 「思考法」に限定、個別銘柄推奨しない |
| 景品表示法 | 「絶対儲かる」等の表現を排除 |
| 競合の参入 | 独自ブランド強化・コミュニティで差別化 |
| コンテンツのネタ切れ | 3本柱×時事ネタで無限展開 |
