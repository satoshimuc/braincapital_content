# Capital Brain — コンセプト設計書

## 1. ブランドコア

### ブランド名
**Capital Brain（キャピタルブレイン）**

### タグライン
**「AIで、資本家の脳をインストールせよ。」**

### サブコピー案
- 給料だけで生きる時代は終わった。
- 思考OSを変えれば、お金の流れが変わる。
- 労働者脳 → 資本家脳へのアップデート。

## 3. コンテンツの3本柱

### Pillar 1: 資本思考（Capital Mindset）
お金に対する「考え方」を根本から書き換える

**キーメッセージ:**
- 「稼ぐ」ではなく「仕組みを作る」
- 消費者 → 投資家 → 資本家の3段階
- 複利思考をすべてに適用する

**コンテンツ例:**
- 「年収500万でも10年で資産5,000万を作る思考法」
- 「AIに聞いた"お金持ちが絶対やらないこと"が深すぎた」
- 「給料の使い方で人生が決まる｜Capital Brainの法則」

### Pillar 2: AI実践術（AI Leverage）
AIツールを使って資産形成を加速する具体的手法

**キーメッセージ:**
- ChatGPTは「相談相手」ではなく「分析エンジン」
- AIで投資判断の精度を上げる方法
- 情報収集の自動化で時間を資産に変える

**コンテンツ例:**
- 「ChatGPTで企業分析を10分で完了させる方法」
- 「AIに自分の家計を分析させたら衝撃の結果が出た」
- 「投資初心者がAIを使うべき5つの理由」

### Pillar 3: 収益構築（Revenue Architecture）
知識を収益に変える情報発信の設計図

**キーメッセージ:**
- 学んだことを発信すること自体が資産になる
- X → note → LINE → 高単価商品の導線設計
- AI×発信で「知識の複利」を回す

**コンテンツ例:**
- 「noteで月5万円を稼ぐまでの全ステップ」
- 「AIを使って1日30分でX運用する方法」
- 「知識を売る"仕組み"の作り方｜0→1完全ガイド」

## 4. ブランドトーン & マナー

### 言葉のトーン
| 要素 | 方向性 |
|------|--------|
| 知的レベル | 専門用語は使うが必ず噛み砕く |
| 語り口 | 断定的・自信がある（「〜かもしれません」は使わない） |
| 距離感 | 先を歩く先輩ポジション（上から目線ではない） |
| 感情 | 煽りすぎない、でも危機感は持たせる |
| 独自性 | 「脳のOS」「思考のアップデート」等のメタファーを多用 |

### NGワード・表現
- 「簡単に稼げる」「誰でもすぐに」→ 煽り系は使わない
- 「投資助言」に該当する具体的銘柄推奨 → 法的リスク回避
- 「絶対に儲かる」→ 景品表示法・金商法に抵触

### OKワード・表現
- 「思考法」「フレームワーク」「仕組み」
- 「資本家脳」「労働者脳」の対比構造
- 「AIに聞いてみた」「AIで分析した結果」
